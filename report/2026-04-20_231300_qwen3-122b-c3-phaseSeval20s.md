## 前提・目的

直前 Phase S-eval-19session (n=19) は「cool time 4 時間 21 分 = intra-day 長時間帯 cool time 初観測 + ub=1664 pool min 更新 14.293 + ub=1584 2 連続崩壊 → 非崩壊復帰 + ub=1586 peak 1 位復帰 mode_B 単独 1 位 + Welch t 2 ub sig 新類型 (1586+1664)」の 5 大事件同時観測、3 帯モデル継続・5 モード体系（A/B 同率解消 → B 単独 1 位）確立。S19 レポートの ★最重要 TODO 筆頭:

> ★最重要: Phase S-eval-20session 候補 — ub=1584 非崩壊復帰後の連続性 (S19 15.083 → S20 非崩壊 or 再崩壊)、ub=1664 下帯 2 連続 or 中/上帯復帰、ub=1586 mode_B 連続 or 再均衡化、所要 40 分

本 Phase は S19 終了（22:10）から **15 分 13 秒後**の 22:25 開始 → 23:10 終了で第 20 session（S20）を追加。これまでの **261 分 intra-day 長時間帯**から通常帯 (13-15 分) への cool time 復帰という重要観測点も同時に取得、以下を観測した:

1. **ub=1664 下帯 2 連続初観測 + 上昇 Δ=+0.399 内下帯維持** — S19 下帯 14.298 → S20 下帯 **14.697** (Δ=**+0.399**)、**下帯 2 連続は 20 session で S19/S20 ペアが初**、帯下限 14.293 は維持（S19 更新値を下回らず）、下帯 9/20=**45.0%** (+2.9pt、単独 1 位維持)、Welch t=**−7.00 sig**
2. **mode_A 復帰で A/B 同率 1 位再均衡 + mode_B 単独 1 位終了** — S19 mode_B 単独 1 位 (7/19=36.8%) → S20 **mode_A (1584,1586,1664) 1 回追加で 7/20=35.0%**、mode_B も S20 でカウント据え置きで 7/20=**35.0%**、**mode_A 7 = mode_B 7 同率 1 位復活**、S18→S19 の「単独 1 位」構造は 1 session で終焉、S20 は 1584 peak 1 位 (7/20=35.0%)
3. **Welch t 3 ub sig 回帰 (+7.16 / +7.03 / −7.00)** — S19 の 2 ub sig 新類型 (+1586 / −1664 / 1584 not_sig) から S20 **3 ub sig 完全同時 (+1584 / +1586 / −1664)**、2 ub sig 類型は **S19 1/20=5.0% のまま単発観測**、3 ub sig 類型 **5/20=25.0%** に増加（S13/S15/S17/S18/S20）、すべて同一「1584+1586+符号 vs 1664−符号」のモード
4. **ub=1586 Δ=+0.023 最小動揺 + S18/S19 連続大幅上昇終了** — S18 +0.275 → S19 +0.257 → S20 **+0.023** (S19 の 1/11 まで減衰)、20 transitions 中最小の絶対値（従来最小は S3→S4 −0.012 の 0.023）、2 session 連続上昇モメンタム消失、ub=1586 15.287 は pool mean 15.146 から +0.141 (S19 +0.126 から +0.015 拡大、2 session 連続正偏差)
5. **ub=1584 15.301 confirmed 復帰 + S13/S15/S17/S18 の 4 session 連続崩壊帯からの完全離脱** — S13 14.693 / S15 13.964 / S17 14.814 / S18 14.798 (4 session 中 3 崩壊) → S19 15.083 復帰 → S20 **15.301 confirmed** (ref 15.293、Δ=+0.008、verdict=confirmed)、20 session で ub=1584 confirmed は S8/S11/S20 の 3 回目、崩壊頻度 5/20=**25.0%** (S19 26.3% → −1.3pt、Wilson CI 11.2-46.9%)
6. **cool time 15 分 13 秒 = 通常帯復帰 + S19 長 cool time 効果の解除** — S19 終了 22:10 → S20 開始 22:25 で **15 分 13 秒**、S18→S19 の 261 分から通常帯 (13-14 分帯、S15-S18) への復帰、**長 cool time (S19) vs 通常帯 cool time (S20) の比較観測点確立**、S20 は 3 ub すべて Δ|≥0.023|〜|0.399| で S19 (|≥0.257|) より縮小（特に ub=1586 +0.023 は 2 ub 停滞型）
7. **pool 差 1586-1584 縮小** — S19 +0.090 → S20 **pool 15.146 vs 15.061 = +0.085** (符号反転維持、絶対値 **0.005 縮小**、3 session 連続拡大から縮小転換)、ub=1586 pool 1 位維持も差はわずか
8. **pool max / pool min 更新なし** — ub=1584 (S2 15.474) / ub=1586 (S15 15.495) / ub=1664 (S17 15.400) / 1664 min (S19 14.293) すべて維持、S20 は pool 極値とは距離あり（1584 15.301 vs max 15.474、1586 15.287 vs max 15.495、1664 14.697 vs min 14.293）
9. **pooled 100-run**: 1584 **15.061** (+0.013) / 1586 **15.146** (+0.008) / 1664 **14.904** (−0.011)、σ_pool 1584 0.344→**0.339 減** / 1586 0.206→**0.203 微減** / 1664 0.304→**0.300 微減**、3 ub とも σ_pool 縮小（S20 の安定 run で分散寄与極小）
10. **warmup1 ub=1584 = 15.333、mode_B_band + mode_C_delta +0.032** — S16/S17/S18/S19 の 4 session 連続「band 14.78-15.37 × delta +0.017 近傍」帯停留に **S20 を加えて 5 session 連続**、ただし S16 非崩壊/S17 崩壊/S18 崩壊/S19 非崩壊/S20 非崩壊（3 非崩壊 2 崩壊）で「warmup 帯 ⇔ 崩壊」連動は否定的補強継続
11. **ub=1664 帯構造 20-session 更新**: 下帯 9/20=**45.0%** (+2.9pt、S20 追加) / 中帯 7/20=35.0% / 上帯 4/20=20.0% (−1.1pt)、**下帯が単独 1 位維持 + 差 +2 session 拡大**、帯下限 14.293 維持（S20 14.697 は下帯 7/9 中央値近く）
12. **compute buffer 20 session 完全一致**: ub=1586 で CUDA0=980.36 / CUDA1=452.31 / CUDA2=452.31 / CUDA3=1558.12 / Host=235.48 MiB（**20 session 全完全一致**）
13. **within-σ 20-session 最小**: ub=1584 σ=**0.003** (S17 0.003 と同値、20-session 最小値タイ)、ub=1586 σ=0.004 / ub=1664 σ=0.005、3 ub 全 ≤ 0.005 で安定性最高、**1 位復帰かつ A/B 再均衡 session は within-σ 最小**の新傾向

### Phase S-eval-20session の核心発見（サマリ）

**結論 1: ub=1584 非崩壊連続性 + confirmed 復帰成立**
- S19 15.083 → S20 15.301 (Δ=+0.218)、2 連続非崩壊、崩壊帯吸引力 2 session 上限仮説補強
- S20 で confirmed 復帰 (ref 15.293、Δ=+0.008)、ub=1584 20 session で 3 回目の confirmed (S8/S11/S20)

**結論 2: ub=1664 下帯 2 連続初観測 + Markov 下帯自己遷移 p(下|下) = 2/8**
- S19 14.298 → S20 14.697 (Δ=+0.399、同帯内上昇)、下帯 2 連続 episode は 20 session で S4-S5 / S19-S20 の 2 episodes
- pool min 14.293 (S19) 維持、帯下限非更新、S20 は下帯中央値近く (14.604 平均 +0.093)

**結論 3: mode_A 復帰 + A/B 7/20 同率 1 位再均衡、mode_B 単独 1 位は S19 単発**
- mode_A (1584,1586,1664): 7/20 = 35.0%
- mode_B (1586,1584,1664): 7/20 = 35.0%
- mode_C (1664,1584,1586): 2/20 = 10.0% (S6/S17)

- mode_E (1586,1664,1584): 2/20 = 10.0% (S13/S15)
- 未観測 (1584,1664,1586): 0/20 (Wilson CI 上限 16.1%)

- Welch sig 類型: **0 ub sig 13/20=65.0%, 1 ub sig 1/20=5.0%, 2 ub sig 1/20=5.0%, 3 ub sig 5/20=25.0%**
