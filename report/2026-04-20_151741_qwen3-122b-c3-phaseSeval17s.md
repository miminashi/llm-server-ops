### Phase S-eval-17session の核心発見（サマリ）

本 Phase は S13-S16 の 4 session 連続の「ub=1586 peak 1 位連覇 + ub=1664 bimodal 周期 2 + ub=1584 崩壊-回復 cycle」を**1 session で 3 軸同時破綻**させる激変 session。S6 以来 11 session ぶりの mode_C 再発、ub=1664 pool max 更新、ub=1586 17-session 最深崩壊の 3 事件で 17 session 体系を再編した。

**結論 1: ub=1664 3 帯モデル確定（下帯 41.2% / 中帯 41.2% / 上帯 17.6%）**
- 下帯 < 14.80: S1/S4/S5/S7/S9/S14/S16 (7 session)
- 中帯 14.80-15.20: S2/S3/S10/S11/S12/S13/S15 (7 session)
- 上帯 > 15.20: S6/S8/**S17** (3 session、平均 15.356)
- 「ub=1664 peak 1 位 ⇔ 上帯 session」完全連動 (3/3=100%)

**結論 2: peak order 5 モード体系再確認（A/B 同率 + C/E 同率 + D 孤立）**
- mode_A (1584, 1586, 1664): 6/17 = 35.3%
- mode_B (1586, 1584, 1664): 6/17 = 35.3%
- mode_C (1664, 1584, 1586): 2/17 = 11.8% (**S6/S17** で 2 回目)
- mode_E (1586, 1664, 1584): 2/17 = 11.8% (S13/S15)
- mode_D (1664, 1586, 1584): 1/17 = 5.9% (S8)
- 未観測 (1584, 1664, 1586): 0/17 (Wilson CI 上限 18.4%)

**結論 3: ub 別独立変動モデル 再強化**
- S17 で ub=1584 下降 (-0.324) + ub=1586 下降 (-0.411) + ub=1664 上昇 (+0.804) の 2 ub 同方向 + 1 ub 逆方向
- 「共通 drift」仮説 完全否定（S16 の逆方向同時に続く 2 例目）

**結論 4: ub=1584 崩壊「2 session 周期」仮説**

- 「時間減衰 + 2 session 周期」複合モデル
