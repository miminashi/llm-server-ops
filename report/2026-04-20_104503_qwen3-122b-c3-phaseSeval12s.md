3. **ub=1664 中帯 (S2, S3, S11 の 3 点) + 中-下間 (S10 の 1 点) の帯分離確定**（S12 でどの帯に落ちるか）

## 再現性分析と解釈

### 1. peak order mode A（1584, 1586, 1664）の 6 度目、独走拡大

S12 で mode A 順序が 6 度目出現、mode B (4/12) との差 **+2** で独走拡大。6 可能順序中:
- **mode A (1584, 1586, 1664): 6 / 12 (S1-S3, S9, S11, S12)** ← 独走
- mode B (1586, 1584, 1664): 4 / 12 (S4, S5, S7, S10)
- mode C (1664, 1584, 1586): 1 / 12 (S6)
- mode D (1664, 1586, 1584): 1 / 12 (S8)

残 2 種類 (1584, 1664, 1586) / (1586, 1664, 1584) は 12 session で依然未観測（Wilson 95% CI P(中位) ∈ [0%, 26.5%]、11 session 28.5% から縮小）。12 session で **「ub=1664 中位忌避」性質が更に強化**、S11 で観測された ub=1664/1586 差 +0.010 の接近は S12 で -0.338 まで拡大、**中位接近は一時的で「2 値状態性（1 位 or 最下位）」に復帰**。

### 2. ub=1664 中帯一過性仮説の浮上（中帯滞在 1 session 限定）

S11 で確定した ub=1664 「4 モード分布」（下 5/11, 中-下間 1/11, 中帯 3/11, 上帯 2/11）は、S12 で下帯再出現（14.721）により **「中帯滞在は 1-3 session 限定」** という新仮説が浮上:

- 中帯滞在期間: S2-S3（2 session 連続、Δ=+0.093 で内部遷移）→ 後続 S4 で下帯復帰
- 中帯再出現: S11（1 session）→ S12 で下帯復帰（-0.317）
- **「中帯に 2 session 以上連続で留まる」**は S2-S3 の初期 1 例のみ、それ以降は単発

Δ=+0.093 帯遷移ステップ検証:
- S2→S3 +0.093（中帯内小振幅移動）
- S10→S11 +0.093（中-下間 → 中帯下限、帯間遷移）
- S11→S12 **-0.317**（中帯下限 → 下帯、-0.31 級の大振幅で **同じ Δ 値で反復せず**）

**Δ=+0.093 ステップは「中帯内振幅の特徴値」** と再解釈、「帯間遷移の固有ステップ」説は **S11→S12 で反証**。中帯は n=12 で 3/12 = 25% 出現、**「上帯 + 中帯 = 5/12 ≒ 42%」が閾値超え頻度**で「下帯 + 中-下間 = 7/12 ≒ 58%」崩壊頻度と整合。
