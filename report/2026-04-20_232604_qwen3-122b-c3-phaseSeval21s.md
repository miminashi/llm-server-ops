## 前提・目的

直前 Phase S-eval-20session (n=20) は「cool time 15 分 13 秒 = 通常帯復帰 + ub=1664 下帯 2 連続初観測 14.697 + mode_A 復帰で A/B 同率 7/20 再均衡 + Welch 3 ub sig 回帰 + ub=1586 Δ=+0.023 最小動揺 + ub=1584 confirmed 復帰 15.301」の 5 大事件同時観測。S20 レポートの ★最優先 TODO 群:

1. **ub=1664 下帯 2 連続 session の「3 連続 or 脱出」検証**
2. **ub=1586 Δ=+0.023 最小動揺 session の再現性**
3. **ub=1584 3 連続非崩壊 + confirmed 継続の定着**
4. **mode_A/mode_B 均衡 steady-state の定着**
5. **3 ub sig 類型 5/20=25.0% 頻度の Welch 非独立性**

本 Phase は S20 終了（23:10）から **15 分 35 秒後**の 23:26 開始 → 00:12 終了で第 21 session（S21）を追加。通常帯 cool time 2 session 連続継続（S19→S20 15'13" + S20→S21 15'35"）という観測点も同時に取得、以下を観測した:

1. **ub=1664 下帯 2 連続 → 上帯復帰 + 下帯 3 連続 episode 不成立** — S20 下帯 14.697 → S21 **15.280** (Δ=**+0.583**、上帯 15.20 超え)、**下帯 2 連続 episode は S19/S20 で終結、3 連続は不成立**、上帯 21 session で 5 回目 (S6/S8/S17/S18/S21)、帯下限 14.293 (S19) 維持、**下帯吸引力 2 session 上限仮説補強**（S4-S5 / S19-S20 の 2 episode × 2 sessions = 4 sessions のみ、3 連続 0 回）
2. **mode_E (1586, 1664, 1584) 3 回目観測 + mode 分類 21-session 再均衡** — S13/S15 に続き S21 **mode_E**（3/21=**14.3%**）、mode_A (1584,1586,1664) 7/21=33.3% / mode_B (1586,1584,1664) 7/21=33.3% の A/B 均衡 7/7 維持、mode_C/D 2/21=9.5% 同率、未観測 mode (1584,1664,1586) 0/21 のみ
3. **Welch 2 ub sig 対称 subtype (+1586 / +1664 / −1584 not_sig)** — S19 の 2 ub sig 類型 (+1586 sig / −1664 sig / +1584 not_sig) **鏡像 subtype** で 2 ub sig 類型 **2/21=9.5%** に増加、「1584 not_sig + 1586 × 1664 同方向 sig」サブ類型が 2 種（正方向 S21 / 負方向 S19）で対称観測、3 ub sig 類型 5/21=23.8% は据え置き
4. **ub=1586 Δ=+0.090 小動揺継続、Δ=+0.023 最小動揺は S20 単発** — S20 +0.023 → S21 **+0.090** (3.9 倍拡大)、|Δ|≤0.03 の「停滞 session」は ub=1586 で S3→S4 -0.012 / S11→S12 +0.011 / S19→S20 +0.023 の 3/20 transitions のみ、S20→S21 で非継続、「3 ub sig 回帰 + A/B 再均衡 session は Δ 最小」仮説は否定
5. **ub=1584 非崩壊 3 連続成立 15.025 (崩壊帯 +0.025 ギリギリ、confirmed 不成立)** — S19 15.083 → S20 15.301 → S21 **15.025**、3 連続非崩壊成立（崩壊帯脱出）だが S21 15.025 は崩壊しきい値 15.0 から +0.025 と極めて近く、**崩壊再突入の可能性**を示唆、confirmed (ref 15.293 ± 0.05) 達成は S20 のみ、S21 Δ_1run=−0.268 で reject (14.693 〜 S13 崩壊帯同程度)
6. **cool time 15 分 35 秒 = 通常帯 2 session 連続継続** — S20 終了 23:10 → S21 開始 23:26、通常帯 cool time（13-16 分）2 連続確立、S19 (261 分) 長 cool time 効果は S20 で解除・S21 で完全解除、S21 |Δ_max|=0.583 (ub=1664) は S20 |Δ_max|=0.399 から +0.184 拡大（cool time 類似でも変動幅は session 依存、単純な cool time 縮小 → 変動縮小勾配は否定）
7. **pool 差 1586-1584 拡大再転** — S20 +0.085 → S21 **pool 15.157 vs 15.059 = +0.098** (符号反転維持、絶対値 **0.013 拡大**、S19 +0.090 水準を超過)、ub=1586 pool mean 1 位維持 + 差 +0.098 で pool 幅拡大
8. **ub=1664 pool mean 更新** — S20 14.904 → S21 **14.922** (+0.018)、pool max 15.400 (S17) / pool min 14.293 (S19) は維持、σ_pool 0.300 → **0.303 微増**（上帯復帰で分散寄与増大）
9. **pooled 105-run**: 1584 **15.059** (−0.002) / 1586 **15.157** (+0.011) / 1664 **14.922** (+0.018)、σ_pool 1584 0.339→**0.331 減** / 1586 0.203→**0.204 微増** / 1664 0.300→**0.303 微増**、ub=1584 は σ_pool 縮小（S21 15.025 が pool mean 近く寄与）
10. **warmup1 ub=1584 = 15.036、mode_B_band + mode_C_delta +0.011** — S16-S20 の 5 session 連続「band 14.78-15.37 × delta +0.017 近傍」帯停留に **S21 を加えて 6 session 連続**（S16 非崩壊/S17 崩壊/S18 崩壊/S19 非崩壊/S20 非崩壊/S21 非崩壊、4 非崩壊 2 崩壊）で「warmup 帯 ⇔ 崩壊」連動の否定的補強継続、warmup1 15.036 ≒ eval 15.025 で Δ=+0.011 は Phase 最小級（S6 +0.017 相当）
11. **ub=1664 帯構造 21-session 更新**: 下帯 9/21=**42.9%** (S20 45.0% から −2.1pt、下帯新規追加なし) / 中帯 7/21=33.3% (据え置き) / 上帯 5/21=**23.8%** (+3.8pt、S21 追加)、**下帯 1 位維持、上帯は 2 連続 (S17/S18) → 下帯 2 連続 (S19/S20) → 上帯 1 連続 (S21) で帯間振動観測**
12. **compute buffer 21 session 完全一致**: ub=1586 で CUDA0=980.36 / CUDA1=452.31 / CUDA2=452.31 / CUDA3=1558.12 / Host=235.48 MiB（**21 session 全完全一致**）
13. **within-σ 21-session 最小タイ**: ub=1586 σ=**0.002**（S20 ub=1584 σ=0.003 と並ぶ最小タイ、21-session 最小値）、ub=1584 σ=0.007 / ub=1664 σ=0.008、3 ub 全 ≤ 0.008 で安定性最高級、**mode_E session × 2 ub sig 対称 subtype 条件下で ub=1586 within-σ 最小**の新傾向
