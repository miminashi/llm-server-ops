直前 Phase S-eval-18session (n=18) は「ub=1584 崩壊 3 session 連続 (S15/S17/S18、2 session 周期仮説完全否定) + ub=1664 上帯 2 連続 (S17/S18、18 session 初記録) + mode_D 10 session ぶり 2 回目 + Welch t 3 ub 同時 sig 2 session 連続 (S17/S18)」の 4 大事件同時観測で、3 帯モデル継続・5 モード体系（A/B 同率 + C/D/E 同率）確立。S18 レポートの ★最重要 TODO 筆頭:

### Phase S-eval-19session の核心発見（サマリ）

本 Phase は S18 の「ub=1584 崩壊 3 連続 + ub=1664 上帯 2 連続 + mode_D 10 session ぶり再発 + Welch t 3 ub sig 2 session 連続」の 4 大事件同時観測から 4 時間 21 分後（14-18 倍の cool time ）、予期された「正常域停留」ではなく **5 仮説同時破綻 + 2 ub sig 新類型 + ub=1664 pool min 更新 + mode_B 単独 1 位** という高エントロピー結果を示した。19 session 体系は「1 主要 + 1 準主要 + 3 副 + 1 未観測」の 6 モード分類に進化、長 cool time sensitivity が新仮説として確立された。

**結論 1: ub=1584 4 連続崩壊仮説完全否定、「崩壊帯吸引力 2 session 上限」仮説へ**
- 崩壊-復帰 2-2 サイクル: S15 崩壊 → S16 非崩壊 → S17/S18 崩壊 → **S19 非崩壊**
- 崩壊帯 14.7-14.9 での吸引力は最大 2 session 連続で消滅、復帰 Δ=+0.285 は高速リカバリ
- 崩壊頻度 5/19=26.3% に低下、Wilson CI 48.8% で幅依然大

**結論 2: ub=1664 上帯 3 連続仮説完全否定 + pool min 更新、下帯単独 1 位復活**
- 上帯 4 session (S6/S8/S17/S18) で cluster 終結、S19 下帯急落 Δ=−0.991 (17 transitions 最大)
- pool min 14.293 は過去 S1 14.580 から −0.287 更新、下帯極値レンジ拡張
- 下帯 8/19=42.1% 単独 1 位、中帯 7/19 との差 +1 session

**結論 3: ub=1586 peak 1 位復帰 + mode_B 単独 1 位への構造転換**
- mode_A (1584,1586,1664): 6/19 = 31.6%
- **mode_B (1586,1584,1664): 7/19 = 36.8% NEW 単独 1 位**
- mode_C (1664,1584,1586): 2/19 = 10.5% (S6/S17)
- mode_D (1664,1586,1584): 2/19 = 10.5% (S8/S18)
- mode_E (1586,1664,1584): 2/19 = 10.5% (S13/S15)
- 未観測 (1584,1664,1586): 0/19 (Wilson CI 上限 16.8%)

**結論 4: Welch t 2 ub sig 類型新規観測**
- Welch sig 類型: **0 ub sig 13/19=68.4%, 1 ub sig 1/19=5.3%, 2 ub sig 1/19=5.3% NEW, 3 ub sig 4/19=21.1%**
- S19 (+1586 sig / −1664 sig / 1584 not_sig) は 19 session で初の 2 ub sig
- |t_1664|=22.57 は 19 session 中最大絶対値、下帯急落の統計的極端性

**結論 5: cool time 261 分 = intra-day 長時間帯 cool time 初観測 + 大変動連動**
- S18 終了 17:03 → S19 開始 21:24 = 4 時間 21 分 (18-19 倍の cool time)
- 3 ub とも |Δ|≥0.257 の大変動 session、「長 cool time → 大変動」実証観測
- Welch 類型で 2 ub sig 初観測も長 cool time 効果の可能性
- cool time 13 分 (S16: 1ubsig) / 14 分 (S14-S18: 0-3 ubsig) / 261 分 (S19: 2ubsig) の分布で中帯 13-14 分は session 依存、長帯 261 分は独立変動強化

**結論 6: ub 別独立変動モデル 3 session 連続強化**
- S17: +1584 sig / −1586 sig / +1664 sig (混在)
- S18: −1584 sig / −1586 sig / +1664 sig (混在)
- S19: +1584 not_sig / +1586 sig / −1664 sig (混在 2 ub sig)
