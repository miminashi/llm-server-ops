## 補足

### Phase S-eval-18session の核心発見（サマリ）

本 Phase は S17 の「3 ub sig 大変動 + ub=1664 上帯 pool max 更新 + ub=1586 17-session 最深崩壊」の直後 session で、期待された「正常域復帰」ではなく **3 仮説同時破綻 + 3 ub sig 2 session 連続 + 上帯 2 連続 + mode_D 再発** という高エントロピー結果を示した。18 session 体系は「2 主要モード + 3 副モード + 1 未観測モード」の 6 モード分類に進化、Markov 解析の基礎が整った。

**結論 1: ub=1584 崩壊「2 session 周期」完全否定、連続崩壊帯仮説へ**
- 崩壊間隔単調減少: S4→S13=9 → S13→S15=2 → S15→S17=2 → **S17→S18=1**
- S17→S18 Δ=-0.016 は 17 transitions 中最小絶対値、崩壊帯 14.7-14.9 が吸引子化
- 崩壊頻度 5/18=27.8% に上昇、Wilson CI 上限 50.9%

**結論 2: ub=1664 上帯 2 連続 = 18 session 初記録、Markov 自己遷移観測**
- 上帯 4 session (S6/S8/**S17/S18**)、S17→S18 で p(上|上)=1/3 の推定値（S6→下、S8→下、S17→上）
- 上帯出現時は peak 1 位 ub=1664 の完全連動 4/4=100% を維持
- ub=1664 pool mean 14.950 に対し、上帯平均 15.339 は +0.389 の顕著な分離

**結論 3: peak order 5 モード体系 A/B + C/D/E 3 副同率へ均衡化**
- mode_A (1584,1586,1664): 6/18 = 33.3%
- mode_B (1586,1584,1664): 6/18 = 33.3%
- mode_C (1664,1584,1586): 2/18 = 11.1% (S6/S17)
- **mode_D (1664,1586,1584): 2/18 = 11.1% (S8/S18) 新、10 session ぶり再発**
- mode_E (1586,1664,1584): 2/18 = 11.1% (S13/S15)
- 未観測 (1584,1664,1586): 0/18 (Wilson CI 上限 17.6%)

**結論 4: 3 ub sig 大変動 2 session 連続 (S17/S18) = 新パターン**
- Welch sig 類型: 3 ub sig 4/18=22.2%, 1 ub sig 1/18=5.6%, 0 ub sig 13/18=72.2%
- S13/S15/S17/S18 の 3 ub sig 4 回のうち、S17→S18 が初の連続、「大変動継承」仮説
- 「大変動 → 収束」から「大変動 → 大変動」へのパターン拡張

**結論 5: ub 別独立変動モデル再強化 + ub=1586 崩壊 1 session 限定仮説復活**
- S18: ub=1584 微減 (-0.016) + ub=1586 上昇 (+0.275) + ub=1664 微減 (-0.108)
- 3 ub 同方向否定 2 session 連続（S17 も逆方向混在）
- ub=1586 は S17 崩壊 → S18 即復帰で「崩壊 1 session 限定」保持、ub=1584 は 3 連続で否定継続、**ub 別に崩壊ダイナミクスが異なる** 2 session 再強化

**結論 6: cool time 14 分帯の大変動率 75%、13 分帯の収束率 100%**
- 14 分帯 (S14=0, S15=3ubsig, S17=3ubsig, S18=3ubsig) = 3/4 大変動
- 13 分帯 (S16=1ubsig) = 1/1 収束
- cool time 13/14 分の 1 分差が session 変動パターンに強く影響、検定候補
