## 前提・目的

直前 Phase S-eval-4session (n=4) は (1) ub=1584 が S4 で −0.728 大崩壊、(2) ub=1664 が「3 session 単調増加」から S4 で S1 以下まで戻り bimodal/periodic の可能性、(3) ピーク順序が S4 で崩壊し ub=1586 が新たな 1 位、(4) S4 全 ub が prior pool 対比 significant 下振れの「session group 低速側」、(5) ub=1586 のみが 4 session で唯一 partial_drift 維持、を報告した。

その最重要 TODO 筆頭として **「Phase S-eval-5session（n=5 で σ_session 収束、ub=1584 崩壊頻度、ub=1664 周期性確認）」**（所要約 50 分）が残置されていた。

本 Phase は **時間を空けた第 5 session（S5）を追加**し、以下を達成する:

1. σ_session（セッション間標準偏差）の n=5 推定と収束性確認
2. **ub=1584 崩壊頻度のカウント**（S1-S5 中で eval_mean < 15.0 となった session 数）
3. **ub=1664 bimodal/periodic 仮説の検定**（5 session 時系列の Δ 符号交互性、上昇/下降頻度）
4. ピーク順序 S4 → S5 の連続性（ub=1586 ピークが 2 連続か）
5. pooled 25-run mean / σ の算出（特に ub=1584 / ub=1664 の収束性確認）
6. S4 共通下振れ（session group 低速側）の S5 再現性または乖離
7. compute buffer の 5 session MiB 単位一致の再確認

### 判定しきい値

- **fully_independent**: 5-session range (max−min) ≤ 0.02 t/s
- **partial_drift**: range ≤ 0.10 t/s
- **session_dominated**: range > 0.10 t/s
- **ub=1584 崩壊判定（本 Phase 新規）**: eval_mean < 15.0 t/s

## 再現性分析と解釈

### 1. ub=1584 大崩壊は単発イベント、頻度 20%

S4 の −0.728 大崩壊は S5 で再現せず。崩壊頻度: 1/5 = 20%。**「崩壊 session」は session 切替時のレアイベント** で、頻度が 1/5 と低いため n=5 でも分布形状の確定は困難。確率モデル（5 session 中 1 回崩壊 → 確率 0.2）の信頼区間 (Wilson 95% CI: [3.6%, 62.5%]) は広く、**真の崩壊頻度推定には n=10-20 必要**。

S4 と S5 の差: cool time S4 14 分 vs S5 11 分（S4 → S5 はさらに短い）にもかかわらず S5 では崩壊なし。**cool time が崩壊の決定因子ではない**ことが示唆される。

### 2. ub=1664 bimodal/periodic 仮説は棄却、上昇 + 単発ダウンスパイクパターン

5 session の Δ パターン: + + − +
- 完全交互（bimodal/periodic）: NO
- 単調増加: NO （S4 で逆方向）
- 「単調増加 + S4 単独ダウンスパイク」が最も説明力の高い解釈

S4 を外れ値として除く 4 session（S1, S2, S3, S5）でも: 14.646 → 15.042 → 15.135 → 14.714 と単調ではない（S5 で軽い後退）。**ub=1664 の長期傾向はまだ確定不能**、S6 以降の継続観測必要。

### 3. ub=1586 が初めて session_dominated に転落、最ロバスト称号崩壊

S1-S4 で唯一 partial_drift を維持していた ub=1586 が、S5 で 15.321 と +0.187 上振れ → 5-session range 0.187 で session_dominated 突入。

|  | 4-session (n=4) | 5-session (n=5) |
|---|---|---|
| range | 0.070 | **0.187** (+167%) |
| σ_session | 0.033 | **0.074** (+124%) |
| pooled σ (n) | 0.030 (n=20) | **0.068** (n=25) (+127%) |
| verdict | partial_drift | **session_dominated** |

「ub=1586 は最ロバスト」性質は **n=4 の偶然の一致** だった可能性が高い。ただし、ub=1586 の pooled σ 0.068 は ub=1584 (0.295) と ub=1664 (0.224) の **1/3〜1/4 で依然最も小さい**ため、相対的には最ロバストな性質を維持。「絶対 session_dominated だが相対的に最も安定」という新しい位置づけ。

### 4. ピーク順序: S4-S5 で ub=1586 連続 1 位

ピーク 1 位の出現頻度: ub=1584 60% (3/5), ub=1586 40% (2/5), ub=1664 0%。**S1-S3 と S4-S5 で 2 つのモードに分かれる**:

- **モード A (S1-S3)**: ub=1584 が 1 位、ub=1586 と接近気味（差 +0.018, +0.266, +0.213）
- **モード B (S4-S5)**: ub=1586 が 1 位、ub=1584 とは S4 で離れ（+0.503）S5 で接近（+0.114）

S6 以降でモード A に戻るか、モード B が継続するかが鍵。確率的には A=3/5, B=2/5 で A 優勢だが、最近 2 session が B のため遷移確率の可能性も。

### 5. S4 共通下振れは S5 で再現せず

S4 の「全 ub prior pool 対比 significant 下振れ」は S5 で再現せず: ub=1584 not_sig +0.040, ub=1586 significant +0.153, ub=1664 significant −0.140。**S4 の「session group 低速側」は S4 固有の現象**で、共通要因（thermal / 他プロセス）由来の確証は得られず。S4 のみ崩壊した可能性が強まる。

### 6. S4-S5 間隔 11 分（5 session 最短 cool time）が崩壊回避

S5 は S4 終了 (04:02) から 11 分後 (04:13) に開始。これは 5 session 最短 cool time。にもかかわらず S5 では ub=1584 崩壊せず、ub=1586 はむしろ +0.153 上振れ、ub=1664 は緩和 (S4 14.593 → S5 14.714)。**短い cool time は性能低下の必要条件ではない**。

### 7. warmup ub=1584 +0.31 普遍則の完全棄却

warmup 1 - eval mean Δ:
- S1: +0.304
- S2: +0.306
- S3: +0.309
- S4: +0.153
- **S5: +0.164**

**S4-S5 連続で Δ ≈ +0.16**、S1-S3 の +0.31 帯から大幅縮小。**「+0.31 普遍則」は完全に棄却**、新たに **「+0.16 帯（S4-S5）」** と **「+0.31 帯（S1-S3）」** の 2 モードが見える。これも S1-S3 と S4-S5 のモード分離仮説を支持。
