4. **warmup ≒ eval 現象 (S9 で 3 ub 全体拡大) の再現性** — session 単位切替機構の未解明
5. **warmup1 absolute 第 4 帯** (S7/S8/S9 3 連続再現、15.40-15.44 中心) の継続確認

8. **ub 別独立変動の 10 session 継続支持**

本 Phase は **S9 終了から 15 分後の第 10 session（S10）を追加**し、以下を達成する:

1. ub=1586 崩壊後 1 session での大振幅復帰（+0.457）観測、S6→S7 パターンの 2 周期目確定
2. peak order **mode B（1586, 1584, 1664）順序の 4 度目出現**（mode A と 1 位タイ）、残 2 種類は依然未観測
3. ub=1664 S10 14.945 で **60% 崩壊**（6/10）到達、3 帯構造から外れる中間値として帯定義の再検討
4. warmup ≒ eval 現象は ub 別に独立（ub=1584 通常復帰 / ub=1586・1664 継続）
5. warmup1 ub=1584 absolute 15.340 → mode_B band 復帰、第 4 帯連続 4 回には至らず
6. pooled 50-run 統計確定、σ_pool は 3 ub 全部で n=45→50 で縮小
7. **Welch t で ub=1664 が初の not_sig**（10 session 中 29/30 significant、ub=1664 S10 が初例外）

## 再現性分析と解釈

### 1. peak order mode B（1586, 1584, 1664）の 4 度目、mode A と 1 位タイ

S10 で mode B 順序が 4 度目出現し、mode A と 4/10 で 1 位タイ。6 可能順序中:

### 2. ub=1664 帯モデルの再定義

S9 で提案した「下帯 14.59-14.74 (5/9) / 中帯 15.04-15.14 (2/9) / 上帯 15.29-15.38 (2/9)」モデルを S10 が覆す:
