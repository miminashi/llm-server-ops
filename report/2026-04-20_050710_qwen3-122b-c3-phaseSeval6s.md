## 前提・目的

直前 Phase S-eval-5session (n=5) は (1) ub=1584 S5 復帰（14.631→15.207、崩壊頻度 1/5=20%）、(2) ub=1664 bimodal 棄却、上昇+単発ダウンスパイク型、(3) ub=1586 初めて session_dominated 転落（range 0.070→0.187、+167%）、(4) S1-S3 / S4-S5 モード分離仮説提示、(5) warmup +0.31 普遍則棄却（S4-S5 で +0.16 帯）、を報告した。

その最重要 TODO 筆頭として **「Phase S-eval-6session（モード分離仮説検証、ub=1586 転落継続判定、ub=1664 反転継続判定）」**（所要約 50 分）が残置されていた。

本 Phase は **S5 終了から 16 分後の第 6 session（S6）を追加**し、以下を達成する:

1. **ub=1586 の S6 挙動で転落原因切り分け**: S5 上振れ (+0.153) が偶発か継続ドリフトか
2. **ub=1664 反転継続 vs 単独ダウンスパイクの判別**
3. **ub=1584 崩壊頻度 n=6 更新**（Wilson 95% CI 絞り込み）
4. **S1-S3 / S4-S5 モード分離の S6 検証**（S6 がどちらのモードに属するか）
5. **pooled 30-run mean / σ**
6. **warmup1 ub=1584 absolute 帯のモード判定**
7. **ピーク順序 6-session 集計**（1 位頻度比の確定度向上）

## 再現性分析と解釈

### 1. S6 でモード分離仮説（mode A / mode B）が棄却、3 モード以上の存在を示唆

- mode_A (S1-S3 平均): ub=1584=15.345, ub=1586=15.179, ub=1664=14.941
- mode_B (S4-S5 平均): ub=1584=14.919, ub=1586=15.227, ub=1664=14.653
- **S6**: ub=1584=15.289, ub=1586=**14.727**, ub=1664=**15.292**

S6 は mode_A / mode_B どちらにも該当しない「モード C」候補。**時間帯依存モデルだとすれば 03:00 以前 (A) / 03:00-05:00 (B) / 05:00 以後 (C) の 3 フェーズ**。あるいは **モード分類自体が不適切**で、各 ub が独立に session 別変動する可能性。

### 2. ub=1586 の「最ロバスト」性質は完全に反転

|  | 4-session (n=4) | 5-session (n=5) | 6-session (n=6) |
|---|---|---|---|
| range | 0.070 | 0.187 | **0.594** |
| σ_session | 0.033 | 0.074 | **0.204** |

S6 で ub=1586 は 14.727 と初めて崩壊 (< 15.0)、pooled σ 0.189 は ub=1584 (0.272) の 70% に達し、「最ロバスト」性質の完全反転。

### 3. ub=1664 が 6 session で初めて 1 位、ピーク順序の恒常性崩壊

5 session 連続で 3 位だった ub=1664 が S6 で 1 位 (15.292) へ。**ub=1664 は決して「遅い ub」ではなく、session 状態により最速にもなりうる**ことが確定。ピーク 1 位頻度比は 3/6/50% (1584) : 2/6/33% (1586) : 1/6/17% (1664) だが、6 session では **決定的なピーク ub は特定できない**。

### 4. ub=1584 崩壊頻度は 1/6=16.7%、Wilson 95% CI [3.0%, 56.4%] に縮小

n=5→6 でやや CI 縮小。依然推定幅広く、n=10-20 での絞り込み必要。**ub=1586 でも S6 で崩壊発生** → 崩壊現象は ub=1584 固有ではなく **ub によらず session 依存**。

### 5. S4 と S6 は 2 種類の異常 session パターン

- **S4 型異常**: 全 ub が prior pool 対比 significant 下振れ（session group 低速側）
- **S6 型異常**: ub 別に逆方向の大幅ゆらぎ（ub=1586 激減、ub=1664 激増、ub=1584 平均）

**共通要因を疑った S4 型に対し、S6 型は ub 固有要因**。session 状態が複雑で、単一のグローバル要因（thermal / DRAM 一括）で説明不可能。

### 6. S5-S6 間隔 16 分（3 番目の長さ）でも大ドリフト

S5-S6 間隔 16 分は 6 session 中 3 番目（最短 S4-S5 11 分 < S3-S4 14 分 < S5-S6 16 分 < S1-S2 17 分 < S2-S3 18 分）。にもかかわらず S6 で最大規模のドリフト発生。**cool time と性能ドリフトに単調相関なし**を追加的に確認。

### 7. ub=1664 は S4 単独下落を除けば「単調上昇」仮説が復活

- S1→S2: +0.395
- S2→S3: +0.093
- S4 除外して連続視: S3→S5: −0.421 or S3→S6: +0.157
- S5→S6: +0.579

S4 を外れ値として除いた S1+S2+S3+S5+S6 の 5 点では 14.646 → 15.042 → 15.135 → 14.714 → 15.292。S3→S5 で戻ったが S5→S6 で過去最高へ更新。**長期的には上昇傾向可能性**。

### 8. warmup1 ub=1584 Δ の新モード (+0.017) 発見

過去 5 session の warmup1 Δ は +0.30-0.31 (mode_A) / +0.15-0.16 (mode_B) の 2 帯。S6 の Δ=+0.017 は両帯に該当せず、新たな「**Δ ≈ 0 モード**」。warmup 効果がほぼゼロになる session が存在、新たな切り分け軸。
