## 前提・目的

直前 Phase S-eval-7session (n=7) は (1) S7 ピーク順序 mode B 復帰で S6 mode C 単発と判明、(2) ub=1586 bouncing 機構確定（S5→S6→S7 で ±0.6 振幅）、(3) ub=1664 「散発的 outlier 型」確定、(4) **ub 別独立変動モデルが決定的支持**（S7 で 3 ub の最近接モードがすべて異なる）、(5) ub=1584 / ub=1586 崩壊頻度 1/7=14.3% / ub=1664 崩壊頻度 4/7=57.1%、(6) warmup1 ub=1584 absolute 15.418 の第 4 帯出現、を報告した。

その最重要 TODO 筆頭として **「Phase S-eval-8session（ub 別独立モデル再確認、ub=1664 spike 頻度絞り込み、ub=1586 bouncing 周期性検証、warmup1 第 4 帯再現性、pooled 40-run σ_pool 安定性）」**（所要約 40 分）が残置されていた。

本 Phase は **S7 終了から約 22 分後の第 8 session（S8）を追加**し、以下を達成する:

1. **ub 別独立モデルの S8 再確認**（S7 で 3 ub 最近接モードが異なる事実の再検証）
2. **ub=1664 spike 頻度の n=8 絞り込み**（4/7 → 4/8 or 5/8、Wilson 95% CI 絞り込み）
3. **ub=1586 bouncing 周期性**（S7 +0.407 復帰後の S8 挙動: 周期ならば再下落、ランダムならば微動）
4. **ub=1584 崩壊頻度 n=8 更新**
5. **warmup1 ub=1584 absolute 第 4 帯（15.418）再現性**
6. **pooled 40-run mean / σ の継続観測**（bouncing 効果の n=35→40 変化）
7. **σ_pool/σ_run_avg 倍率の 8-session 推移**

## 再現性分析と解釈

### 1. ピーク順序 mode D（1664, 1586, 1584）の初出現

S8 で新パターン mode D（1664, 1586, 1584）が観測された。6 つの可能順序中 **4 種類が出現**:
- mode A (1584,1586,1664): S1-S3
- mode B (1586,1584,1664): S4, S5, S7
- mode C (1664,1584,1586): S6
- mode D (1664,1586,1584): S8

残 2 種類: (1584,1664,1586) / (1586,1664,1584) は未観測。ub 別独立モデルの下では「どの順序も等確率」というわけではないが、**「ub=1664 が 1 位になる場合、2 位は ub=1584 (S6) または ub=1586 (S8) の 2 パターン」** が確認された。

### 2. ub=1664 は「3 baseline 帯」モデルへ

S8 で ub=1664 が **15.380 の過去最高値** を記録したことで、ub=1664 の分布は 2 帯ではなく 3 帯に拡張の可能性:

- **下帯（14.59-14.74）**: S1, S4, S5, S7（4 session）
- **中帯（15.04-15.14）**: S2, S3（2 session）
- **上帯（15.29-15.38）**: S6, S8（2 session）← **mode_C_band = 15.3 付近**

「14.5-14.9 が baseline、> 15.0 が稀な spike」という S7 モデルは、**S8 で上帯 2 session 目に突入した結果、「3 帯構造（下 / 中 / 上）」として再解釈する必要がある**。

### 3. ub=1586 の bouncing 継続・振幅縮小

S7 +0.407 復帰に続き S8 +0.139 微増。bouncing の振幅は S5→S6 の 0.594 から徐々に減衰中:

| Δ区間 | 絶対値 |
|---|---|
| S4→S5 | 0.187 |
| S5→S6 | 0.594（最大崩壊） |
| S6→S7 | 0.407（最大復帰） |
| S7→S8 | 0.139（微上昇） |

振幅減衰は「bouncing oscillation + noise」の典型パターン。**周期性（S8 で再下落）は棄却**、代わりに「減衰振動 + ノイズ床」モデルが成立。

### 4. ub=1584 は安定中で特異変化なし

S7 → S8 +0.139 微上昇、絶対値 15.260 は 8 session 中 range 中位（2nd quartile）。ub=1584 の崩壊は S4 1 度のみ、pooled σ 0.237 は低下傾向。ub=1584 が最も「drift が抑制されている」ub。

### 5. ub=1664 warmup = eval 現象

S8 で **ub=1664 warmup1 ≈ warmup2 ≈ eval_mean**（15.398 / 15.395 / 15.380）。従来の S1-S7 では warmup1 が eval より 0.03-0.15 速いのが常態だったが、**S8 では warmup「加速効果」が消失**。ub=1664 の spike session では warmup 初期から既に定常状態に到達している可能性。

### 6. cool time 22 分でも大 drift 発生（S7→S8）

S7-S8 cool time 22 分は **8 session 中最長**。これまでの 14-21 分レンジを初めて超えたが、**ub=1664 で +0.643 の史上最大 drift を観測**。cool time-drift 無相関の仮説を再確認、むしろ「cool time が長いほど state が deeper に変化する」可能性を残す（ただし n=1 では断言不可）。

### 7. within-session σ は引き続き極低値

S8 の within-session σ:
- ub=1584: 0.004 (相対 0.026%)
- ub=1586: 0.003 (相対 0.020%)
- ub=1664: 0.001 (相対 0.007%) ← **8 session 中最小**

ub=1664 σ=0.001 は「spike session 中は within σ がさらに小さくなる」挙動か。pooled-to-within 倍率 102.8x は 3 ub 中最大、session-level drift と within-session noise の規模差が **100 倍** を超えた。

### 8. ub 別独立モデルの再補強

S7 で 3 ub の最近接モードが異なる事実から「ub 別独立モデル」を結論したが、S8 でも 3 ub 最近接モードが (1584→C / 1586→B / 1664→C) と **S7 と異なるパターン** で混在。4 種類の peak order × 2 の近接モード分布 = ub 別独立変動が session 越しに継続。

### 9. 計測プロトコル自体の確度

8 session 連続で:
- 起動失敗 0 件
- run 失敗 0 件
- predicted_n=256 完走率 100%
- compute buffer MiB 一致率 100%

計測プロトコル自体は完全に頑健。
